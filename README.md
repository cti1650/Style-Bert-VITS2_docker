# Style-Bert-VITS2

## このリポジトリについて

知人が「感情豊かな音声合成ができると噂の Style-Bert-VITS2 を試してみたい」とのことでその実行方法などの共有用に試してみたリポジトリです。

## ローカルサーバー

```
http://localhost:8000
```

## コマンド

### サーバー起動

```
docker compose up
```

### サーバー停止

```
docker compose stop
```

### サーバー破棄

```
docker compose down
```
